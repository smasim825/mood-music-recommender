# Music Mood Recommender

## Overview
Imagine you are reading a book and want to listen to a song that matches the mood of the text. This application:

1. Extracts text from an image using **OCR** (Tesseract).
2. Detects the **emotion** of the text (happy, sad, angry, calm, romantic, fear, neutral) using **TextBlob**.
3. Recommends songs dynamically using the **YouTube Music API** based on the detected emotion.

---

## Features
- Text extraction from images using OCR
- Emotion detection with **7 categories**
- Dynamic music recommendations from YouTube Music
- Randomized results to avoid repetition
